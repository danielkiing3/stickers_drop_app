## Installation

To run this app locally, follow these steps:

1. Clone the repo locally

```bash
git clone https://github.com/danielkiing3/stickers_drop_app.git
cd stickers_drop_app
```
2. Install dependencies:
```bash
flutter pub get
```
3. Run the app:
```bash
flutter run
```
